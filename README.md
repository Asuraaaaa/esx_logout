**esx_logout** - Simple logout system for Public use.

**Description**

A Character logout system for the dependency esx_kashacters.

**Dependency**
esx_kashacters

**INSTALLATION**
* Put the  **esx_logout**  resource in your FiveM/resource folder 
* Start it using **ensure esx_logout**
